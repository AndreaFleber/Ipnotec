# 🎵 Ipnotec

> For those who think in notes.

A hypnotic screensaver for musicians and score readers. Fluid staves, calligraphic notes, slow infinite scroll, optional ambient drone.

**Live:** https://ipnotec-music.netlify.app
**itch.io:** https://andrea-fleber.itch.io/ipnotec

---

## Features

- 5 layered staves with wave motion
- Calligraphic notes — quarter, half, eighth, whole, slurs, breath marks, jazz slashes
- Ambient drone (open fifth A) — activate with a click anywhere
- Pure CSS animations — never pauses, works on mobile
- PWA installable on Android home screen

## How to use

Open https://ipnotec-music.netlify.app in any browser.

On Android: Chrome menu → Add to home screen.

Click anywhere to toggle the ambient drone.

## Tech

HTML · CSS animations · Web Audio API · No dependencies · No frameworks
